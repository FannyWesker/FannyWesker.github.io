This is a practice project on film presentation and search. Do not good enough, please exhibitions.

先来个功能介绍：分类的排序、搜索功能、详情的搜索、分页设置。
这次的练习里，获取到的接口里，有些照片是没有的，没有做这个细节处理。做个判断能够解决这个问题。
【🐰效果效果效果，点这里…】

在这次的训练项目中，用的主要是jQuery，写的很繁杂的感觉，还是需要用其他的js框架。了解过了AngularJS，但是深入使用还没有过，所以下一个小训练项目将会以AngularJS写。

在这次训练项目中遇到过一些问题，也有许多收获，使我印象深刻的有以下几点（顺序不分先后）：

  1. 要使两个<div>重叠，可以用css的 position 搭配 Z-index使用，必须得同时使用。
  2. Bootstrap的dropdown组件，会出现按钮与显示的下拉菜单width不一样，可以设置.dropdown-menu{ min-width:100%; }就好了。
  3. 公用接口，一般是需要申请一个key，这次项目是用的“The Movie Database API”这里再补充两个公用API：“1.Baidu Map” “2.zhihu”
